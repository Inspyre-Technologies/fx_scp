What is SCP?

SCP (Secure Copy Protocol) is a protocol that allows secure transferring of files between two computers via a network connection. It utilizes SSH (Secure Shell) for both connecting and securing that connection.

In Unix (Linux, FreeBSD, etc) this is a command that can be used to inititialize a secure copy between machines. However, it’s only available via the command line interface (or terminal emulator) and can require a bit of Unix experience to get working properly or even become aware of it’s availablity in your PATH. Here’s an example of how one would copy a file utilizing SCP over a network connection:

$>  scp -v taylor@remote.machine:/home/taylor/Documents/Writings/Proofread/salems_pot.epub -p 24 taylor@remote2.machine:/home/taylor/Documents/Writings/for_submittal/salems_pot.epub

Issuing the above command on Unix machine would cause the machine to reach out and connect to remote.machine using the username taylor

What is FXRuby?

Ruby is a programming language, like C++, JavaScript, or Python. And Fox is a GUI toolset written in C++ so that programmers have a library of references to use when creating their own program that they want to add a GUI (graphical user interface) to.

FXRuby is a collection of tools a Ruby programmer can use to utilize the Fox toolkit without having to leave the comfort of their Ruby environment. In a way, FXRuby is a bridge between Ruby and the Fox Graphical Toolkit.

What is FX::SCP?

FX::SCP aims to use FXRuby to create a simple yet robust GUI (Graphica User Interface) for users to utilize when making use of the native scp command in Unix (a Windows version will come).
